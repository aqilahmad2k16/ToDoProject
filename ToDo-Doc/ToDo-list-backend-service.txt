**Get** `/tasks/{id}`
- get task by id

```json
[
	{

		"id": 3,
		"title": "Hey there! hope doing well",
		"due_date": 23122022,
		"completed": false,
		"notes": [
		
		]
	}
]
````

| query param | definition|
|-------------| ----------|
|`notes`      | if true, task has notes else not |

examples - 
`/tasks/3?notes=true` - fetch detaisl of task id = 3 and including it's notes
